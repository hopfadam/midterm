Hello World in Python

Description:

This code is a simple Python script that defines and calls a function to print "Hello World!" to the console. It contains a `Hello` function that, when called, executes the `print("Hello World!")` statement. The script also defines a `main` function that calls the `Hello` function. The `if __name__ == "__main__":` check ensures that `main()` is called only when the script is executed directly, not when imported as a module in another script, demonstrating a structured way to organize code in Python for readability and reusability.
