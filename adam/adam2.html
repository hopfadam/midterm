Hello World in C++

Description:

This code is a basic C++ program that prints "Hello World!" to the console using the `std::cout` object and the `std::endl` manipulator from the iostream library for output. The `main` function serves as the entry point for the program and returns 0, indicating successful execution. This example showcases the fundamental structure and output method in a C++ program.
